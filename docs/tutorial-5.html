Functions

Functions are reusable blocks of code.

Example:

def greet(name):
    print(f"Hello, {name}!")

def add(a, b):
    return a + b

greet("Alice")
result = add(5, 3)
print(result)

Key Points:

  • Use 'def' to define a function
  • Functions can accept parameters
  • Functions can return values
  • Call a function by using its name with parentheses
